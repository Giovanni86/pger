A partir de esta explicación, conformé tres grupos para que cada uno evalúe una dimensión de la gestión de recursos que tiene la institución. Al finalizar, cada grupo expuso su trabajo.

Para esta actividad es recomendable utilizar o proyectar las pautas de evaluación incluidas en la Matriz de Gestión de Recursos.

Luego, en conjunto, se proponen las mejoras en las prácticas desarrolladas en la institución. Para esto, cada grupo podrá identificar un aspecto central a mejorar proponiendo acciones concretas para fortalecer y mejorar.
