Para iniciar este camino

Para iniciar este camino es necesario que conozcamos la estructura del curso
Este curso consta de cuatro módulos de formación

Módulo 1. Relación con el entorno y el direccionamiento estratégico
Módulo 2. Direccionamiento estratégico
Módulo 3. Liderazgo y participación para un direccionamiento estratégico efectivo
Módulo 4. Organización estratégica y orientación al cambio

Al interior de cada módulo encontraremos Unidades de Aprendizaje que nos permitirán aproximarnos a una conceptualización y llevarlo a la práctica.

Como recurso adicional encontraremos la Bitácora de trabajo. Cuando encuentre este icono podrá recurrir a este material para registrar su proceso.
